                                                                •	Personal Budget Tracker 
Managing personal finances has become increasingly challenging as modern lifestyles involve
frequent digital transactions, diverse spending categories, and variable income sources.
Traditional methods such as handwritten logs, spreadsheets, and basic mobile apps often fail to
provide real-time insights or meaningful analysis. To address these limitations, this project
introduces an AI-Integrated Personal Finance Tracker Application, built using React Native
and Expo Go, which combines intelligent automation with intuitive design to empower users to
take control of their daily financial activities.

The primary objective of this application is to simplify expense tracking, enhance financial
awareness, and support better decision-making through the integration of artificial intelligence.
The app allows users to record daily transactions, categorize expenses, monitor income, and set
personalized financial goals. Unlike conventional finance trackers, this system uses LLM-based
AI models to automatically interpret user-entered descriptions, categorize expenses, generate
predictive insights, and provide personalized recommendations that help users reduce
unnecessary expenses and achieve long-term savings targets.

This project demonstrates the practical integration of AI into a mobile finance management
system, showcasing how natural language processing and data analysis can significantly enhance
user experience. The application not only assists users in capturing financial information
accurately but also helps them understand their behavior through trend analysis, automated
categorization, and personalized insights. By combining goal-oriented features with real-time
financial monitoring, the system promotes consistent financial discipline and encourages users to
stay aligned with their long-term objectives.
